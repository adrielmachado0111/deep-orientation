## requerimientos

Librerías necesarias para el funcionamiento del script inference.py:

- python 3.6.*
- opencv-python==3.4.3.*
- tensorflow==1.12.3
- tqdm
- scipy
- seaborn
- scikit-learn
- matplotlib

python inference.py
